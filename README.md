# Generador de referencia APA 7 para artículos científicos

Tu asistente personal para que la bibliografía de tus investigaciones o clases deje de ser un dolor de cabeza. Esta extensión extrae metadatos directamente desde el navegador y genera una cita perfecta en **APA 7ª edición** al instante.

## ✨ Características

* **Privacidad Total:** No envía datos a servidores externos ni requiere inicio de sesión. Todo el "scraping" y formateo ocurre localmente.
* **Formatos a Medida:** Configura desde la página de opciones si prefieres:
    * **Rich Text:** Para pegar en Word o Google Docs conservando las cursivas.
    * **Texto Plano:** Sin formato extra.
    * **Markdown:** Usando `_cursivas_` para editores como Obsidians o Notion.
* **Discriminación Inteligente:** Si entras a un sitio que no es un paper (como un blog o red social), la extensión te avisará que el formato no está implementado para evitar citas erróneas.

## 📚 Sitios Compatibles (Confirmados)

La extensión está optimizada para leer etiquetas de metadatos académicos (*Highwire Press* y *Dublin Core*), funcionando con alta precisión en:

* ✅ **PubMed**
* ✅ **Springer Nature**
* ✅ Sitios que utilicen estándares abiertos de metadatos científicos.

## 🚀 Instalación

1.  Descarga el código de la extensión en una carpeta local.
2.  Abre Google Chrome y ve a `chrome://extensions/`.
3.  Activa el **"Modo de desarrollador"** (arriba a la derecha).
4.  Haz clic en **"Cargar descomprimida"** y selecciona la carpeta del proyecto.

## 🛠️ Uso

1.  Visita un artículo en PubMed o Springer.
2.  Haz clic en el icono de la extensión (el libro con la flecha).
3.  ¡Listo! La cita se copiará automáticamente a tu portapapeles según tu preferencia.

### Configuración
Para cambiar el formato de salida, haz clic derecho en el icono de la extensión > **Opciones**.
